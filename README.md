## 编译步骤

### 建立开发环境

1. First install grunt

```
$ sudo npm install -g grunt-cli
```

2. Second install uesed plugs 

```
$ cd mywiki/
$ npm install
```

1. run `grunt` to build the project

2. the release images are in `build` directory
